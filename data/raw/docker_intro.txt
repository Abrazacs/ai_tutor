Docker — инструмент контейнеризации приложений.

Контейнер — это изолированная среда на основе образа.

Образ (image):
- Шаблон файловой системы + зависимости
- Многоуровневые слои
- Содержимое описывается в Dockerfile

Основные команды Docker:
- docker pull — загрузить образ
- docker run — запустить контейнер
- docker build — собрать образ
- docker ps — список контейнеров
- docker logs — логи контейнера
- docker exec — выполнить команду в контейнере

Dockerfile — файл инструкции сборки образа:
Пример:
FROM python:3.10
WORKDIR /app
COPY . .
RUN pip install -r requirements.txt
CMD ["python", "main.py"]

Преимущества контейнеров:
- Изоляция процессов
- Лёгкая упаковка приложения
- Быстрый деплой
